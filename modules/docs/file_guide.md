# 각 폴더와 역할

이 문서에선 현재 폴더 아래 파일의 역할에 대해서 설명합니다.

## modules

**폴더 설명:** 프로젝트 전역에서 참조할 수 있는 유틸리티 모듈들입니다.

### _except.py

- 프로그램의 실행 예외를 정의하는 모듈입니다.

### check.py

- 설정 파일을 검사 함수를 제공하는 모듈입니다.

### console.py

- 컬러화된 콘솔 입출력을 제공하는 모듈입니다.

### datatype.py

- 커스텀 데이터 타입을 정의하는 모듈입니다.

### system.py

- 실행환경에 따라 반환값이 변경되는 함수를 제공하는 모듈입니다.

### functions.py

- **여러 파일에서 모듈을 참조하지만, 정확하게 카테고리를 분류하지 못 했을 때 임시로 구현되는 모듈입니다. 파일이 비어있는 것이 제일 좋습니다.**
